## 📖 项目简介

本项目是一个基于成都地铁线路数据开发的查询系统，支持：

- 线路站点查询
- 站点所属线路查询
- 最短路径规划（Dijkstra 算法）

提供两个版本以满足不同使用场景：

- **C++ 版本**：适合开发者和终端用户，轻量高效
- **Web 版本**：现代化网页界面，提供更好的用户体验

---

## ✨ 功能特点

### 核心功能

- ✅ **线路查询** - 查看任意地铁线路的所有站点
- ✅ **站点查询** - 查询指定站点所属的线路
- ✅ **路径规划** - 智能计算两点间最短路径（Dijkstra 算法）
- ✅ **换乘识别** - 自动识别并显示换乘信息
- ✅ **数据完整** - 覆盖成都地铁 1-19 号线及 27 号线（共 14 条线路，292 个站点）

### C++ 版本特点

- 🚀 轻量高效，无额外依赖
- 🖥️ 命令行界面，简洁直观
- 📦 单文件可执行程序
- 💾 内存占用极小
- 🔒 完善的错误处理和输入验证
- 📝 清晰的代码注释和文档
- 🔄 支持循环菜单，方便多次查询

### Web 版本特点

- 🎨 现代化 UI 设计，紫色渐变主题
- 📱 响应式布局，完美支持移动端和平板
- 🔍 站点自动完成提示（datalist）
- ⚡ 快速响应，流畅体验（使用 requestAnimationFrame 优化）
- 🎯 可视化路径展示，换乘站高亮显示
- ♿ 良好的可访问性（ARIA 属性支持）
- 🎭 CSS 变量统一管理主题颜色
- 🛡️ XSS 防护（HTML 转义处理）

### C++ 版本

#### 1. 编译项目

确保已安装支持 C++11 的编译器（如 GCC、MinGW、MSVC）：

**使用 GCC (Linux/Mac/MinGW):**


直接使用 `output/成都地铁路径.exe` 文件

**方式一：运行可执行文件**

### Web 版本

#### 1. 前置要求

- 现代浏览器（Chrome、Firefox、Edge、Safari）
- Python 3.x（用于启动本地服务器）

#### 2. 启动服务器

**方法一：使用启动脚本（推荐）**

1. 双击运行 `启动网页.bat`
2. 脚本会自动：
   - 启动本地服务器
   - 打开浏览器访问 http://localhost:8000

**方法二：手动启动**

```bash
# 进入项目目录
cd 成都地铁路径

# 启动HTTP服务器
python -m http.server 8000

# 在浏览器访问
# http://localhost:8000
```

#### 3. 使用界面

Web 版本提供三个功能标签页：

**1. 线路查询**

- 从下拉菜单中选择线路号
- 点击"查询"按钮
- 查看该线路所有站点的网格展示

**2. 站点查询**

- 输入站点名称（支持自动完成）
- 点击"查询"按钮
- 显示站点所属的所有线路

**3. 路径规划**

- 输入起点站点
- 输入终点站点
- 点击"规划路线"按钮
- 查看详细路径，包括：
  - 总站数
  - 换乘次数
  - 完整路线（换乘站会高亮显示）

#### 4. 注意事项

⚠️ **重要**：不要直接双击 `index.html` 打开！

由于浏览器安全限制（CORS），直接打开 HTML 文件无法加载 JSON 数据。必须通过 HTTP 服务器运行。

✅ **正确方式：**

- 使用 `启动网页.bat`
- 或运行 `python -m http.server 8000`

❌ **错误方式：**

- 直接双击 `index.html`

### C++ 版本

**核心算法：**

- Dijkstra 最短路径算法（优化实现）
- 邻接矩阵存储图结构

**数据结构：**

```cpp
typedef struct {
    char name[50];           // 站点名称
    int line_ids[MAX_LINES]; // 所属线路ID列表
    int line_count;          // 线路数量
    int id;                  // 唯一编号
```

## ❓ 常见问题

### Q1: C++版本运行时提示"找不到路径"

**A:** 确保 `成都地铁路径.txt` 文件在以下位置之一：

- 与可执行文件同一目录
- 可执行文件上级目录

或者将数据文件复制到 `output/` 目录。

### Q2: Web 版本显示"数据加载失败"

**A:** 这是因为直接打开了 HTML 文件导致的 CORS 限制。

**解决方法：**

1. 使用 `启动网页.bat` 启动
2. 或运行 `python -m http.server 8000`
3. 访问 http://localhost:8000

### Q3: 站点名称输入后无结果

**A:** 请确保：

- 站点名称完整（包含"站"字）
- 名称拼写正确
- 站点存在于数据中

### Q4: 如何更新地铁线路数据？

**A:**

1. 编辑 `成都地铁路径.txt` 文件
2. C++版本：直接保存，重新运行
3. Web 版本：运行 `python convert_data.py` 重新生成 JSON

### Q5: 编译时出现错误

**A:** 确保：

- 编译器支持 C++11 标准
- 包含必要的头文件
- 使用命令：`g++ 成都地铁路径.cpp -o 成都地铁路径.exe -std=c++11`
