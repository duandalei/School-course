# 成都地铁查询系统 - Web 版

## 📌 项目简介

这是一个美观实用的成都地铁查询系统网页版，提供了以下功能：

1. **线路查询** - 查看任意线路的所有站点
2. **站点查询** - 查询站点所属的线路
3. **路径规划** - 使用 Dijkstra 算法计算两点间最短路径

## 🚀 快速开始

### 方法一：使用启动脚本（最简单）

1. **双击运行 `启动网页.bat`**
2. 脚本会自动：
   - 启动本地 HTTP 服务器（端口 8000）
   - 在浏览器中打开 `http://localhost:8000`

### 方法二：使用本地服务器（手动）

⚠️ **注意：由于浏览器安全限制（CORS），不能直接双击打开 `index.html`！**

必须通过 HTTP 服务器运行，方法如下：

**Python 3:**

```bash
python -m http.server 8000
```

然后在浏览器访问 `http://localhost:8000`

**Node.js (需要先安装 http-server):**

```bash
npm install -g http-server
http-server -p 8000
```

访问 `http://localhost:8000`

📖 详细启动方法请参考 [`启动方法.md`](启动方法.md)

## 🎯 使用示例

### 线路查询

1. 点击"线路查询"标签
2. 选择要查询的线路（如"线路 1"）
3. 点击"查询"按钮
4. 查看该线路的所有站点列表

### 站点查询

1. 点击"站点查询"标签
2. 输入站点名称（如"天府广场"）
3. 点击"查询"按钮
4. 查看该站点所属的线路

### 路径规划

1. 点击"路径规划"标签
2. 输入起点和终点站点
3. 点击"规划路线"按钮
4. 查看详细路线点的路线信息

## 💡 使用提示

- 🌐 **浏览器要求** - 建议使用现代浏览器（Chrome 90+、Firefox 88+、Edge 90+、Safari 14+）
