### ✅ **方法1：使用 Python（推荐）**

**最简单的方式：**

```
1. 按 Win+R，输入 cmd，回车
2. 进入项目目录：
   cd /d "D:\项目\课程设计\成都地铁路径"
3. 启动服务器：
   python -m http.server 8000
4. 保持窗口打开
5. 打开浏览器访问：
   http://localhost:8000
```

**或者使用批处理文件：**

直接双击 `启动网页.bat` 文件，会自动启动服务器并打开浏览器。

**端口管理：**

```bash
# 查看端口占用
netstat -ano | findstr :8000

# 检查端口状态（示例输出）
# TCP    127.0.0.1:8000    0.0.0.0:0    LISTENING    1234

# 关闭占用端口的进程（替换 1234 为实际 PID）
taskkill /PID 1234 /F
```

### ✅ **方法2：使用 Node.js**

**需要先安装 Node.js（如果未安装）：**

```
1. 打开命令行
2. 进入项目目录：
   cd /d "D:\项目\课程设计\成都地铁路径"
3. 安装 http-server（只需安装一次，全局安装）：
   npm install -g http-server
4. 启动服务器：
   http-server -p 8000
5. 打开浏览器访问：
   http://localhost:8000
```

------

| 项目         | `npm install -g http-server`                          | `npm install http-server`             |
| ------------ | ----------------------------------------------------- | ------------------------------------- |
| 安装范围     | 全局（系统）                                          | 本地（项目）                          |
| 命令可用性   | 任何地方都能用                                        | 只能在项目中用（或用 `npx` 临时运行） |

| 查找命令     | `npm root -g` / `npm prefix -g` / `where http-server` | `npm root` / `npm list`               |

### ⚙️ 【常见错误】

1. **确认 Python 已安装**

   - 命令：

     ```
     python --version
     ```

   - 如果显示版本号（如 `Python 3.11.6`），说明安装成功。

   - 如果报错 `'python' 不是内部或外部命令`，表示未安装或未加入系统 PATH。

2. **确认在正确目录**

   - 启动服务器前，必须进入包含网页文件的目录（有 `index.html`）。
   - 否则浏览器访问时会显示“目录列表”或 404。

3. **确认端口未被占用**

   - 若已有其他程序（例如之前启动的 Python 服务器）使用同一个端口（如 8000），新的服务器就无法启动。

   - 可改用其他端口号（如 8888）：

     ```
     python -m http.server 8888
     ```

   - 或关闭旧的服务器。

------

### ❗【常见错误及解决方案】

| 错误类型  | 错误提示                                                  | 产生原因                       | 解决方法                                                     |
| --------- | --------------------------------------------------------- | ------------------------------ | ------------------------------------------------------------ |
| **错误1** | `'python' 不是内部或外部命令`                             | 系统找不到 Python 路径         | 重新安装 Python，并在安装时勾选“Add Python to PATH”          |
| **错误2** | 浏览器中出现 `ERR_CONNECTION_REFUSED` 或 `无法访问此网站` | 服务器窗口关闭或程序退出       | 重新打开命令行，进入项目目录后再次运行 `python -m http.server 8000` |
| **错误3** | “Address already in use” 或 “端口被占用”                  | 该端口已被其他程序使用         | 关闭占用端口的程序，或使用其他端口号，如 `python -m http.server 8888` |
| **错误4** | 页面显示空白或 404                                        | 当前目录没有网页文件或路径错误 | 确保在包含 `index.html` 文件的目录运行命令                   |
| **错误5** | 页面样式不生效（CSS 无效）                                | 引用路径错误或大小写不一致     | 检查 HTML 中 CSS/JS 文件路径是否正确、区分大小写             |
